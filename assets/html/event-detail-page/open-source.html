Greetings from Innovo Infinity,

Open Source With Scratch, you can program your own interactive stories, games, and animations — and share your creations with others in the online community.

The ability to code computer programs is an important part of literacy in today’s society. When people learn to code in Scratch, they learn important strategies for solving problems, designing projects, and communicating ideas.

Topics that has to be cleared in this session will comprise of:

>> What is OpenSource and GSoC.

>> How to start and proceed working from scratch on OpenSource.

>> How to expand your portfolio and knowledge using OpenSource and contributing to OpenSource?

>> The world of possibilities in GSoC , how to prepare and its advantages?
